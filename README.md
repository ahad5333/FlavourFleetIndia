## âš™ï¸ Getting Started

### Prerequisites

* Node.js (v12+ or latest)
* npm / yarn
* Database installed (e.g., MongoDB, MySQL)
* Git

### Installation

```bash
# Clone the repository
git clone https://github.com/ahad5333/FlavourFleetIndia.git
cd FlavourFleetIndia

# Frontend dependencies
cd frontEnd
npm install

# Backend dependencies
cd ../backend
npm install
```

### Running Locally

#### Backend

```bash
cd backend
npm start  # or `node index.js`
```

#### Frontend

```bash
cd frontEnd
npm start  # or `npm run dev`
```

Access the app at: [http://localhost:3000](http://localhost:3000)
